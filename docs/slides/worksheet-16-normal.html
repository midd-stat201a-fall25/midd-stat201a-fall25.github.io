Percentile problems

Suppose SAT scores are Normally distributed with mean 1100 and standard deviation 200.

  1. Edward earned a 1030 on his SAT. We will find his percentile using code in two ways.
    1. First, draw a picture representing what we want to find. Then write code to find the percentile.
    2. Find Edward’s z-score. Write another line of code to find the percentile.
  2. What is the 97.5th percentile for SAT scores?
    1. Write code to answer this question.
    2. Write a different line of code to answer this question that involves a z-score.
  3. Unrelated to SAT scores: consider the standard normal \(N(0,1)\) distribution. The 25th percentile of this distribution is -0.67. Without doing any work beyond drawing a picture, what is the 75th percentile of the distribution?

Some practice problems

  1. In a law school class, the entering students averaged 160 on the LSAT. The variance was 64. The histogram of LSAT scores followed the normal curve reasonable well.

    1. About what percentage of the class scores below 152?

    2. One student was 0.5 standard deviations above average on the LSAT. About what percentage of the students had lower scores than he did?

  2. Weights of 10-year-old girls are known to be Normally distributed with mean of 70 pounds and standard deviation of 13 pounds. Find the probability that a 10-year-old girl weighs between 60 and 85 pounds two ways:

    1. Optional, but helpful: draw a sketch of the curve and shade in the region of interest.

    2. Write the probability of interest in \(P()\) form. Then write the R code necessary to find this probability, and actually execute the code to obtain the probability.

    3. Confirm your solution in (b) by transforming to z-scores first, then using code again to obtain the probability.

  3. Consider the same scenario as in 2. Without using any code than what is provided below, find the 60th percentile for the weight of 10-year-old girls.

    qnorm(0.6, mean = 0, sd = 1)

  4. \((^*)\) Suppose body temperatures are Normally distributed with mean \(98.6^\circ\) F and standard deviation of \(0.7^\circ\) F. Assuming this is true, answer the following:

    1. Fevers \(103^\circ\) F or higher are considered dangerous. What fraction of people would be expected to have such high a fever?

    2. According to quick Google search, a range for low-grade fever is between \(99.5^\circ\) F and \(100.3^\circ\) F. What is the probability of having a low-grade fever?

    3. What body temperatures would you consider as unusually low? Briefly explain why.

    4. Provide two intervals that each capture/contain 80% of body temperatures.
